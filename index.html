European Court of Human Rights Database

Explored and analyzed the European Court of Human Rights (ECtHR) database, covering cases from 1960 to 2022. The ECtHR enforces human rights across Europe, with binding rulings on member states. My role involved using QGIS to map violations, creating and supplementing shapefiles for 46 countries. I also used Python, matplotlib, and seaborn to generate interactive graphs that expanded the project's scope, providing deeper insights through data visualizations. I employed Python to convert binary codes into meaningful visual representations of the violations across Europe. We showcased our findings using WordPress, making the insights more accessible and visually impactful.

Visualizing Crime

This project analyzes Seattle crime data from the past two years, aiming to create an interactive choropleth map and dashboard to visualize crime trends across neighborhoods. We developed a dashboard using spatial analysis to reveal crime trends and differences in crime types and rates. My role included creating the choropleth and dot density maps with HTML, CSS, and JavaScript, and using Python for data wrangling and formatting. I also merged the crime data with a JSON file highlighting Seattle neighborhoods and incorporated Google packages to display interactive graphs for a comprehensive presentation.

Geo Tours4U: AMTRAKÂ® Empire Builder Train

GeoTour4U was introduced to our project team with the goal of transitioning information from ECHEO to CLIO. These software are used by organizations in order to create audio tours for individuals who aim to learn more about significant routes while riding the Amtrak. The project scope involved creating a tour from Seattle to Wenatchee and a second from Whitefish to Havre which all together covers 645 miles.

My role in this project was versatile, as I was involved in a lot of data entry for the respective nodes, as well as incorporating aspects of product management to create schemes in figma that orientated our steady development. We were able to create a timeline for goals, as well as creating a more updated instructor manual for new users. Using QGIS, we were able to visualize as well as estimate distance and speed between nodes.

Our final output of this project included creating an audio tour within the new CLIO application.

Link: CLIO Tour Webpage: Whitefish to Havre

Dendrogram: Sodo, Seattle

The dataset for this lab was sourced from the Yelp API on 1/28/2027, focusing on gyms in the SoDo area. The original dataset contained over 150 rows of information, but it has been cropped for clarity. Each gym location typically includes 6 reviews; however, for less popular gyms with fewer reviews, only the available number is shown. Some locations have as few as 2 reviews due to limited popularity. The code for this analysis was created using Google Colab.

Image Processing: Color of Seattle

The process of this lab includes taking 30 pictures periodically in a span of a week. The location we chose as a group was on the campus of University of Washington. The area we covered was behind the water fountain showcasing a field of grass. After capturing all 30 images, we move into the Image Processing in colab. We automated the code so that the code captures the 10 most dominant colors and changes the color values in the format of hex instead of RGB. We capture all the data into a dataframe using the pandas library. The final output includes 300 rows of data showcasing the data, color, number of pixels, and percentage.

Economy Analysis: Exploring Demographics

In this project, I used R to analyze various economic indicators, such as housing costs, poverty by race, and poverty by age. I cleaned and processed data from multiple sources, including CSV files, to create a comprehensive dataset. My role included using libraries like ggplot2 and usmap, I developed interactive visualizations to display trends and comparisons over time. I deployed the app using shinyapps.io to make the visualizations accessible online. The interactive platform allows users to explore different economic dimensions through adjustable inputs, enhancing the understanding of key economic trends.
